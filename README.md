# AI Prog HW

**城市出租车出行数据分析与智能问答系统**

---

## 项目简介

本项目基于纽约市黄色出租车 2023 年 1 月公开数据集（约 300 万条行程记录），完成以下四个模块：

| 模块 | 内容 |
|------|------|
| M1 数据处理 | 数据加载、质量报告（缺失率/异常值）、数据清洗、特征提取（7 个衍生特征） |
| M2 分析可视化 | 6 张分析图表：时间规律、工作日 vs 周末、区域热度 TOP10、车费影响因素、乘客人数分析、平均速度分析 |
| M3 预测模型 | TensorFlow 神经网络 + 随机森林对比，预测区域时段出行需求量，报告 MAE/RMSE |
| M4 智能问答 | 命令行交互，支持 5 种问题类型，接入大模型 API 进行回答 |
